01

Rust SFU on Raspberry Pi

Rust + Tokio + Axum. 단일 바이너리, 외부 의존성 제로. 라즈베리파이에서 직접 빌드하고 scp 하나로 배포합니다. Docker도, JVM도, node_modules도 없습니다.

02

4-Stage Telemetry

SDP 협상(S), 인코더(A), SFU 릴레이(B), Jitter Buffer(C). 4개 구간을 3초 주기로 수집하되, 미디어 처리 성능에는 영향을 주지 않습니다. 관측이 품질을 떨어뜨리면 관측의 의미가 없으니까요.

03

Contract Fact-Check

SFU가 WebRTC 계약을 제대로 이행하는지 8개 항목을 자동 검증합니다. NACK → RTX 히트율, PLI → 키프레임 전달, JB 지연 — 숫자가 O/X를 말합니다.
